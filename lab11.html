Вступ до css

CSS (Cascading Style Sheets) - це код, який ви використовуєте для стилізації вашої веб-сторінки. Як і HTML, CSS насправді не є мовою програмування. Це не мова розмітки - це мова таблиць стилів. Це означає, що вона дозволяє застосовувати стилі вибірково до елементів в документах HTML.
